# SQL Data Analysis Project

## Project Overview
This project demonstrates SQL-based data analysis on a retail sales dataset. The objective was to extract meaningful insights using SQL queries.

## Tools Used
- MySQL Server
- MySQL Workbench
- SQL

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()
- ROUND()

## Tasks Performed
- Retrieved records from the dataset.
- Filtered data using WHERE.
- Sorted data using ORDER BY.
- Grouped records using GROUP BY.
- Calculated total revenue.
- Counted orders by product.
- Analyzed payment methods.
- Identified referral sources.
- Calculated average order quantity.

## Key Insights
- Identified the most ordered product.
- Calculated revenue generated by each product.
- Found the most commonly used payment method.
- Analyzed referral sources.
- Calculated average order quantity.
