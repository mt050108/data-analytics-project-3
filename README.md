# SQL Data Analysis Project

## Project Overview
This project demonstrates SQL-based data analysis using MySQL. The objective was to extract meaningful insights from a retail sales dataset by writing SQL queries for filtering, sorting, grouping, and aggregation.

## Tools Used
- MySQL Server
- MySQL Workbench
- SQL

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()
- ROUND()

## Tasks Performed
- Retrieved all records from the dataset.
- Selected specific columns.
- Filtered cancelled orders.
- Sorted products by total price.
- Counted the total number of orders.
- Identified the most ordered product.
- Calculated revenue generated by each product.
- Calculated the average order quantity.
- Analyzed payment methods.
- Analyzed referral sources.

## Key Insights
## Key Insights

- The most ordered product is **PRINTER**.
- The product generating the highest revenue is **CHAIR**.
- The most commonly used payment method is **ONLINE**.
- **INSTAGRAM** is the most common referral source.
- The average order quantity is **2.9458**.
- No significant outliers were observed in the dataset.
